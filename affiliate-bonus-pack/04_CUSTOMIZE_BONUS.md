# 特典4: あなた用の初動アドバイス用テンプレ

Codexを自分の仕事や発信にどう使えばいいか迷ったときに、最初の一歩を整理するためのテンプレです。

完璧な使い方を決める必要はありません。まずは「最初に何を頼むか」「どんな資料を入れるか」「AGENTS.mdに何を書くか」を小さく決めれば十分です。

## 使い方

下のフォームをコピーして、空欄を短く埋めてください。

そのままCodexに貼ると、自分向けの初動プランを作ってもらえます。

## 記入フォーム

```text
私はCodexを最初に使い始めたいです。
以下をもとに、最初の一歩を軽く整理してください。

1. あなたの仕事や発信内容
-

2. Codexで最初にやりたいこと1つ
-

3. よく扱うファイル
-

4. 参考にしたい自分の過去アウトプット
-

5. 今いちばん詰まりそうなポイント
-

出してほしいもの:
- 向いている使い方
- 最初に作るAGENTS.mdの方向性
- 最初に入れる参考資料
- 最初の依頼文
- つまずきやすい点
```

## 返答を受け取ったあとにやること

Codexから返ってきた内容のうち、全部を採用する必要はありません。

まずは次の3つだけ残してください。

1. 最初に試す作業を1つ
2. AGENTS.mdに入れるルールを3から5行
3. 最初に使う依頼文を1本

## うまく使うコツ

```text
返答が広すぎる場合は、次のように追加で頼んでください。

今すぐ15分でできることだけに絞ってください。
最初に開くファイル、貼るプロンプト、次に確認することを3つだけ教えてください。
```

## 迷ったら

何を書けばいいかわからない場合は、空欄のままでも大丈夫です。

その場合は「まだ決まっていません」と書いてCodexに貼ってください。Codexに候補を出してもらってから選べばOKです。

# Codex 初動スターター特典セット

このフォルダは、Hokato 上で「Codexアプリの教科書」を小池のリンクから購入してくれた方向けの Codex 特典パックです。

目的は、Codex を「買ったけど何から触ればいいかわからない」で止めずに、購入直後の15分で最初の1回を試せる状態にすることです。全部を一気に読む必要はありません。まずは下の順番で、必要なものだけ開いてください。

## まず一番ラクに始めるなら

Codex にまとめて作ってもらいたい場合は、最初に [00_CODEX_SETUP_PROMPT.md](00_CODEX_SETUP_PROMPT.md) を開いてください。

中のプロンプトを Codex にそのまま貼ると、推奨フォルダと初期 md ファイルをまとめて作成できます。

## mdファイルの扱い方

mdファイルは、特別なアプリではありません。Markdownという形式のテキストファイルです。

この特典では、mdファイルを次の用途で使います。

- 説明書として読む
- コピペ用プロンプトを保管する
- Codexに渡すルールや手順をまとめる
- あとから自分用に少しずつ追記する

購入直後は、mdファイルを編集しなくて大丈夫です。

まずは `00_CODEX_SETUP_PROMPT.md` を開き、中の「コピペ用プロンプト」を丸ごとコピーして Codex に貼ってください。

## 手作業で15分だけ進めるなら

1. `02_FOLDER_TEMPLATE.md` を見て、Codex 用の作業フォルダを1つ作る
2. `01_AGENTS_STARTER.md` から、用途に近い AGENTS.md を1本だけ貼る
3. `03_PROMPT_PACK.md` から、今やりたい作業に近い依頼文を1本だけ試す

ここまでできれば、最初のゴールは達成です。自分の用途に寄せたい場合だけ、`04_CUSTOMIZE_BONUS.md` を開いてください。

## おすすめの使う順番

### 最初の日

`02_FOLDER_TEMPLATE.md` と `01_AGENTS_STARTER.md` だけで十分です。

フォルダと AGENTS.md があるだけで、Codex への依頼がかなり安定します。最初からプロンプトを大量に試すより、作業場所と基本ルールを先に作るのがおすすめです。

### 慣れてきたら

`03_PROMPT_PACK.md` から、今やりたい作業に近いものを1本だけ選んでください。

うまくいった依頼文は、自分の作業フォルダの `03_prompts` に保存しておくと、次回から使い回しやすくなります。

### 自分用に寄せたい人向け

`04_CUSTOMIZE_BONUS.md` を使って、仕事や発信内容に合わせた最初の使い道を整理してください。

最初から完璧な環境を作る必要はありません。「自分はCodexを何に使うと便利か」が1つ見えれば十分です。

## 迷ったときの選び方

| 状況 | 開くファイル |
|---|---|
| まとめて自動作成したい | `00_CODEX_SETUP_PROMPT.md` |
| 何から始めればいいかわからない | `02_FOLDER_TEMPLATE.md` |
| Codex の返答が毎回ブレる | `01_AGENTS_STARTER.md` |
| 依頼文が思いつかない | `03_PROMPT_PACK.md` |
| 自分の用途に落とし込みたい | `04_CUSTOMIZE_BONUS.md` |

## この特典の使い方のコツ

- 最初から全部読まない
- AGENTS.md は1本だけ貼って試す
- うまくいった依頼文だけ残す
- 参考資料は最初は3つまでにする
- 成果物と元ファイルを同じ場所に置かない

## 最後に

この特典は、きれいな設定を完成させるためではなく、Codexを今日1回使ってみるための入口です。

まずは `00_CODEX_SETUP_PROMPT.md` を使うか、手作業で `02_FOLDER_TEMPLATE.md`、`01_AGENTS_STARTER.md`、`03_PROMPT_PACK.md` の順に進めてください。

1回でも動かせたら成功です。細かい調整は、そのあとで大丈夫です。

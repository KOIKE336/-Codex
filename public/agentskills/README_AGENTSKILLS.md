# AGENT SKILLS の使い方ガイド

この特典で配っている md ファイルは、`Codex に役割を持たせて使いやすくするテンプレ` です。  
いきなり全部を使う必要はありません。まずは 1 本だけ選んで試すのがおすすめです。

## どう使うのか

1. 使いたい skill を 1 本選ぶ
2. その md にある `AGENTS.md に入れる例` を、必要な分だけ project 直下の `AGENTS.md` に入れる
3. 作業するときに `コピペ用プロンプト` をチャットへ送る

## 役割分担

- `AGENTS.md`
  - 毎回守ってほしい進め方
- `今回のプロンプト`
  - 今日やってほしい具体的な依頼

## 最初におすすめの3本

- まず計画を整理したい
  - `01-planning-agent.md`
- 差分を出す前に確認したい
  - `02-pr-review-agent.md`
- バグ調査で止まっている
  - `03-bug-debug-agent.md`

## 使い方のコツ

- 最初は 1 本だけ入れる
- うまくいったら、その project 専用に少しずつ育てる
- 何本も同時に入れると、役割がぶつかりやすい

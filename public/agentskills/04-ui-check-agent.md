# 04. 見た目の仕上げチェックエージェント

## 何をしてくれるか

画面を見て、情報量、余白、強弱、AIっぽさの有無をレビューする役です。  
仕上げ段階で効きます。

## AGENTS.md に入れる例

```md
## UIチェック&改善エージェント

- UIを確認するときは、まず情報密度と視線誘導を見てください
- ありきたりなAIデザインになっていないかをチェックしてください
- 余白、見出しの強弱、配色、カードのメリハリを具体的に指摘してください
- 改善案は優先順位つきで返してください
```

## コピペ用プロンプト

```text
このUIを確認して、初心者向けの特典ページとして読みやすいかレビューしてください。

見てほしい観点:
- 情報量が詰まりすぎていないか
- 見出しと本文の強弱があるか
- 余白が足りているか
- いかにもAIっぽい平均的デザインになっていないか
```

## 使い方のコツ

- 実装前より、1回作ったあとに使う
- `好み` と `読みにくさ` を分けて指摘させる

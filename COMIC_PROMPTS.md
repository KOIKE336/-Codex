# 導入パネル画像プロンプト集

このファイルは、`最初に全体像をつかむ` セクションの ①〜⑤ 用に、
ChatGPT の画像生成へそのまま貼りやすい形でまとめたプロンプト集です。

## 使い方

おすすめは `文字なし画像` で作る方法です。  
理由は、画像生成で日本語文字まで入れると崩れやすいからです。

流れ:

1. 参照キャラ画像をChatGPTに添付する
2. まず `共通設定プロンプト` を送る
3. そのあと `Scene 1` から `Scene 5` を順番に送る
4. 吹き出しや見出しの文字は、このWebサイト側で重ねる

## 共通設定プロンプト

```txt
添付の参照画像のキャラクターを、今後の全画像で主人公として固定してください。

キャラクター固定条件:
- 黄色い肌
- 短めの紫髪
- 大きな丸い目
- レインボーのタイダイTシャツ
- 青い短パン
- 青いスニーカー
- 親しみやすく、少しコミカル
- 同一人物として毎回ブレないこと

全体の作風:
- 日本の初心者向けAI教材の導入に使う、整理された1コマイラスト
- 白背景ベース
- 明るく信頼感があり、少しワクワクする
- 漫画と図解の中間のような見やすい構図
- 線はやや太め、表情はわかりやすく
- 印刷しても見やすい高コントラスト
- UIやガジェット表現は親しみやすく、近未来すぎない
- 文字は画像に入れない
- 吹き出しも不要。あとから文字を載せやすい余白だけ確保する
- 背景はシンプルだが、場面の意味が伝わる小物やUIは入れてよい

構図条件:
- 縦長ではなく、Webのカードに載せやすい横長または正方形寄り
- 主人公の顔や上半身がしっかり見える
- 1枚で状況が伝わる
- 余白をきれいに残す

出力条件:
- 1枚完結
- 高品質
- 教材の導入にそのまま使える完成度
```

テーマ: 最初の疑問

シーン:

## うまくいきやすい追記

各Sceneのあとに、必要ならこれを1行足してください。

```txt
前回までに作った主人公と同一人物として、髪型・顔・服装・体型を変えないでください。
```

## 文字を入れたい場合の短い追加指示

どうしても画像内に短い見出しだけ入れたい場合は、最後にこれを足します。

```txt
日本語テキストは短い見出し1つだけにしてください。長文は入れないでください。
```

ただし、見た目の安定感は `文字なし` のほうが高いです。
